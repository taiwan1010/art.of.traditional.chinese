Level 0
Secret Code — 注音符號
Taiwan uses its own phonetic system — Zhuyin. 37 symbols + 5 tones. No Roman letters. No shortcuts. Just the code.
0 / 37 symbols learned
21 consonants — the opening sound of a syllable. Each has a visual and sensory memory hook.
Click any symbol to explore it.
Five tones — same sound, five meanings
Every syllable in Mandarin has a tone. Change the tone, change the word entirely. The tone mark is written above the last vowel — or in Zhuyin, placed to the right of the syllable.
Tone on 媽 mā

How syllables are built
Not every combination is allowed. Understanding the rules helps you read and write any syllable correctly.
Consonant types
Type 1 — Must combine
ㄅㄆㄇㄈ ㄉㄊㄋㄌ ㄍㄎㄏ ㄐㄑㄒ
Cannot appear alone. Must be followed by a Semivowel (ㄧㄨㄩ), a Vowel, or both.
Type 2 — Can stand alone
ㄓ ㄔ ㄕ ㄖ ㄗ ㄘ ㄙ
These seven can form a syllable by themselves — no Vowel needed. e.g. ㄗˋ = 字 (character)
Valid combination patterns
Consonant + Vowel
ㄅ + ㄚ = ㄅㄚ
爸 (dad) · 把 (hold) · 八 (eight)
Consonant + Semivowel
ㄅ + ㄧ = ㄅㄧ
比 (compare) · 筆 (pen) · 逼 (force)
Consonant + Semivowel + Vowel
ㄅ + ㄧ + ㄢ = ㄅㄧㄢ
邊 (side) · 變 (change) · 便 (convenient)
Type 2 Consonant alone
ㄗ alone = ㄗ
字 (character) · 自 (self) · 子 (child/son)
Semivowel alone or + Vowel
ㄧ alone = ㄧ  ·  ㄧ + ㄢ = ㄧㄢ
一 (one) · 言 (speech/language)
